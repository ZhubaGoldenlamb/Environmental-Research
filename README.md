# 🌿 Butte Creek Restoration — Invasive Species Monitoring  
### Ulmus pumila (Siberian Elm) — Arizona Riparian System

> ⚡ GIS + Field Ecology + Restoration Systems  
> Tools: QGIS • Trimble GPS • Ecological Field Methods

---

## 📍 Overview

This project evaluates the spread of the invasive tree **Siberian elm (*Ulmus pumila*)** along Butte Creek and assesses the effectiveness of long-term restoration efforts led by the Butte Creek Restoration and Education Council (BCREC).

The work integrates **field ecology, GPS data collection, and GIS analysis** to quantify invasive species density and compare restored vs. unmanaged areas.

## 🎯 Objectives

- Quantify the abundance and spatial distribution of *Ulmus pumila*
- Compare invasive vs. non-invasive tree populations
- Evaluate restoration outcomes across multiple study areas
- Develop a repeatable monitoring framework

---

## 🧪 Methods

### Field Data Collection
- Surveyed three creek quadrants:
  - Upstream (reference ecosystem)
  - Central campus (restored)
  - Lower campus (unrestored)
- Classified individuals into:
  - Seedlings (<1 cm)
  - Saplings (1–6 cm)
  - Mature trees (>6 cm)
- Counted all invasive and non-invasive trees

### Geospatial Workflow
- Collected coordinates using **Trimble Juno GPS**
- Processed data in **QGIS**
- Generated spatial distribution maps

### Data Processing
- Normalized density (trees per 100 m²)
- Compared against historical baseline (1996)

---

## 📊 Key Results

- Invasive dominance reduced from **78% → 38%** in restored areas
- Unrestored zones remained highly invaded (**up to 87%**)
- Native tree populations approached upstream reference conditions
- High seedling density linked to disturbance and irrigation zones

## 🔍 Insights

- Long-term restoration significantly reduces invasive dominance
- Disturbance (roads, development) drives invasion patterns
- Infrastructure (trails, access) indirectly supports restoration success
- Seedling control is critical for long-term management

# 🌊 Estero Sargento Vegetation Mapping  
### Remote Sensing + Field Ecology — Gulf of California, Mexico

> ⚡ Remote Sensing + GIS + Field Expedition  
> Tools: ArcGIS • LANDSAT • TSAVI • GPS • TWINSPAN

---

## 📍 Overview

This project produced a **vegetation classification map of Estero Sargento**, a coastal wetland in Sonora, Mexico.

The work was conducted during a **four-month field station program in the Gulf of California**, involving intensive study of regional **flora, fauna, and coastal ecosystems**.

The estuary lies on **Comcaác Indigenous land** and remains relatively undisturbed, but lacks formal environmental protection.

---

## 🌎 Field Program Context

This project was part of an immersive **field-based environmental studies program**, where we:

- Conducted ecological fieldwork across the Gulf of California
- Studied coastal, desert, and marine ecosystems
- Integrated field observation with GIS and remote sensing workflows
- Lived and worked in remote conditions, collecting primary data

This context shaped the project’s **applied, ground-truthed approach**.

---

## 🎯 Objectives

- Classify vegetation communities using satellite imagery
- Validate classifications through field sampling
- Produce a vegetation map for conservation use
- Contribute to a regional geospatial dataset

---

## 🧪 Methods

### Remote Sensing Workflow
- Acquired **LANDSAT imagery (USGS)**
- Generated **TSAVI (soil-adjusted vegetation index)**
- Performed **unsupervised classification (isocluster)** in ArcGIS

### Field Validation (Ground-Truthing)
- Established **21 relevé plots**
- Recorded species composition and abundance
- Collected GPS coordinates for spatial alignment

### Statistical Analysis
- Applied **TWINSPAN multivariate analysis**
- Refined spectral classes into ecological communities

---

## 📊 Key Results

- Identified **7 vegetation communities**, including:
  - Mangrove forest
  - Cactus scrub
  - Coastal dune systems
  - Halophytic shrublands
- Produced a validated vegetation map
- Identified regions requiring further sampling

---

---

## 🔍 Insights

- Remote sensing requires field validation for ecological accuracy
- TSAVI improves classification in **high bare-soil desert systems**
- Gulf estuaries are **ecologically critical but under-mapped**
- Indigenous-managed lands are key to conservation

---

## 🛠️ Tools & Technologies

- ArcGIS
- LANDSAT imagery
- TSAVI vegetation index
- GPS field collection
- TWINSPAN ecological analysis

---

## 🌎 Impact

This project contributes **baseline geospatial data** for a relatively unprotected estuary and supports **Indigenous-led ecosystem monitoring**.

It demonstrates a scalable workflow for mapping remote ecosystems using limited access field data.
